# NS PRO

NodeSeek 非官方 Android 客户端。WebView 打开 [NodeSeek 官网](https://www.nodeseek.com)，本地注入 Nodeseek Pro 用户脚本，免装油猴。

**Android 7.0+** · 当前版本 **v3.0.18**

## 下载

| 文件 | 说明 |
|------|------|
| [NS-PRO.apk（直链下载）](https://github.com/lmh-codes/NS-PRO/releases/download/v3.0.18/NS-PRO.apk) | 安装包 |

也可在 [Releases](https://github.com/lmh-codes/NS-PRO/releases) 页面下载最新版。

## 功能

- 打开官方 NodeSeek，页面与官网一致
- 内置 Nodeseek Pro：签到、下拉加载、快捷回复、过滤、历史、等级标记、Callout、图片预览等
- 首次安装引导设置链接默认打开方式
- 长按页面打开菜单（刷新、重载脚本、清除登录等）
- 外链走系统浏览器

## 安装

1. 下载 `apk/NS-PRO.apk`
2. 在 Android 手机上允许安装未知来源应用
3. 安装后打开 **NS PRO** 即可使用
